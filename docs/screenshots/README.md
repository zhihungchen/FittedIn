# Screenshots

This directory contains screenshots used in the project documentation.

## Files

- `homepage-hero.png` - Main homepage screenshot displayed in the README.md

## Usage

The homepage screenshot is referenced in the main README.md file:

```markdown
<img src="docs/screenshots/homepage-hero.png" alt="FittedIn Homepage - Transform Your Health Journey" width="900"/>
```

## Adding New Screenshots

When adding new screenshots:
1. Save the image with a descriptive filename (e.g., `homepage-hero.png`)
2. Keep file sizes reasonable (< 500KB recommended)
3. Use PNG format for best quality or JPG for smaller file sizes
4. Update this README if adding new screenshots for documentation
